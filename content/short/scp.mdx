# scp

```bash
scp <file> <destination-username>@<destination-ip>:~<destination-file-location>
```
## Transfer file

Enable SSH server on the destination
```
sudo pacman -S openssh
sudo systemctl start sshd
sudo systemctl enable sshd
```
